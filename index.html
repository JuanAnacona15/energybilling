Energy Billing App

About Application

This application was created in order to demonstrate the developer's knowledge, it is not an application that is used in business or production environments. With this web is intended to demonstrate with a practical example knowledge about software development technologies such as server management, databases, APIs, programming languages that allow to connect the entire system and above all, the ability to design functional systems.

Main Functions

Generate Invoice

The last energy bill can be consulted using the contract number, this will allow the user to download his bill in pdf format with all the billing data.
